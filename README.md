## Getting Started

First, install the `node_modules`:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

## Run Tests

Run all component tests headlessly in Chrome:

```bash
npm run test
```

Open component tests in Chrome:

```bash
npm run test:open
```
